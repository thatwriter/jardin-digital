## Escaleta es el guión del radio
El contenido es: elemento(s), duración, comentarios
eg.:

| Voz         | Música    | FX     | Duración | Comentarios |
| ----------- | --------- | ------ | -------- | ----------- |
| Bla bla bla |           | Lluvia | 10"      | Drama       |
|             | Canción 1 |        | 2:00"    | Fade In     |
|             |           |        |          |             |

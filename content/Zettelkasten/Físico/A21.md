## Operaciones con conjuntos
Unión: 
	$A\cup B\ \epsilon \ A:(a,b,c) \ B: (d,e,f) = (a,b,c,d,e,f)$  


Intersección: 

	$A\cap B \ \epsilon \ A:(a,b,c) B:(c,d,e) =(c)$


Diferencia: 

	$A\setminus B \ \epsilon \ A:(a,b,c) B:(b,c,d) = (a)$ 
	
	$B\setminus A \ \epsilon \ A:(a,b,c) B:(c,d,e) = (d,e)$


Diferencia simétrica: 

	$A \triangle B \ \epsilon \ A: (a,b,c) B:(c,d,e) = (a,b,d,e)$


Complemento: 

	$A' \ \epsilon \ A': (a) =(b,c,...,z)$
	
	$B' \ \epsilon\ B':(b) = (a,c,...,z)$


Conjunto vacío: 

	$A()\ \epsilon \ \varnothing$
